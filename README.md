Website profesional untuk memperkenalkan produk **Clap Bright**, lampu pintar berbasis IoT yang dikontrol dengan tepukan tangan.

---

## 📱 Preview Website

Website ini dirancang dengan pendekatan **Mobile-First** dan **fully responsive** di semua perangkat:

- ✅ **Mobile** (320px - 767px)
- ✅ **Tablet** (768px - 991px)  
- ✅ **Desktop** (992px - 1199px)
- ✅ **Large Desktop** (1200px+)

## 🚀 Cara Menggunakan

### **Setup Awal:**

1. Download file: `index.html`, `style.css`, `script.js`
2. Buka `index.html` di browser

### **Kustomisasi Konten:**

**Mengganti Teks:**
```html
<!-- Edit di index.html -->
<h1 class="display-3 fw-bold mb-4">
    [Edit judul di sini]
</h1>
```

**Menambahkan Gambar:**
```html
<!-- Ganti icon dengan gambar -->
<img src="images/product.jpg" alt="Produk">
```

**Embed Video YouTube:**
```html
<!-- Ganti VIDEO_ID -->
<iframe src="https://www.youtube.com/embed/VIDEO_ID"></iframe>
```

## 🌐 Deploy Website

### **GitHub Pages:**
```bash

git commit -m "Clap Bright website"
git push
# Enable di Settings → Pages
```

### **Netlify:**
1. Drag & drop folder ke netlify.com
2. Website langsung live!

## 📧 Setup Form Kontak

2. Edit `script.js` dengan form endpoint
