Lampu Pintar yang Mengerti Anda

Clap Bright hadir sebagai solusi modern untuk mengontrol pencahayaan rumah Anda. Dengan teknologi sensor suara canggih, lampu dapat dinyalakan atau dimatikan hanya dengan tepukan tangan. Tidak perlu saklar, tidak perlu smartphone, cukup tepuk tangan dan lampu akan merespons.

KEUNGGULAN

Mengapa Memilih Clap Bright?

Fitur-fitur unggulan yang membuat hidup Anda lebih mudah

Sensor Suara Pintar

Dilengkapi sensor suara berkualitas tinggi yang dapat mendeteksi pola tepukan tangan dengan akurasi 95%. Bekerja optimal dalam radius 5 meter.

Konektivitas IoT

Terhubung dengan WiFi untuk monitoring real-time melalui dashboard web. Pantau penggunaan listrik dan status lampu kapan saja, di mana saja.

Kontrol Smartphone

Selain tepukan, Anda juga bisa mengontrol lampu melalui smartphone. Aplikasi sederhana dan user-friendly untuk iOS dan Android.

Jadwal Otomatis

Atur jadwal nyala dan mati lampu secara otomatis. Cocok untuk keamanan rumah saat Anda bepergian atau untuk rutinitas harian.

Sistem Keamanan

Dilengkapi proteksi overload, short circuit protection, dan auto-shutdown untuk keamanan maksimal instalasi listrik rumah Anda.

Ramah Lingkungan

Menggunakan LED berkualitas tinggi dengan konsumsi daya rendah. Membantu mengurangi jejak karbon dan menghemat tagihan listrik.

CARA KERJA

Bagaimana Clap Bright Bekerja?

Teknologi sederhana dengan hasil yang luar biasa

Diagram Cara Kerja

Sensor → Mikrokontroler → Relay → Lampu
1

Deteksi Suara

Sensor suara menangkap pola tepukan tangan dari pengguna dan mengubahnya menjadi sinyal digital.

2

Pemrosesan Sinyal

Mikrokontroler ESP8266 memproses sinyal dan mengidentifikasi apakah tepukan valid (1x atau 2x).

3

Eksekusi Perintah

Relay modul menerima perintah dan mengaktifkan/menonaktifkan aliran listrik ke lampu.

4

Monitoring Cloud

Data penggunaan dikirim ke cloud server untuk monitoring konsumsi energi dan statistik penggunaan.
